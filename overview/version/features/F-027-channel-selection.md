# F-027 通道选择策略

**里程碑**: v0.2.0 (多通道)

## 用户故事

作为发送方，系统自动选择最佳交付通道，通道故障时自动切换备用通道，我无需关心技术细节。必要时我也可以手动指定通道。

## 选择优先级

Agent 直连 > IM Bot > Webhook > 邮件。查询接收方已注册的通道列表后按优先级匹配。

## 故障降级

首选通道发送失败时，自动降级到下一可用通道，托盘通知「{通道名} 发送失败，已切换至 {备用通道} 发送」。

## 手动指定

交付窗口底部增加通道选择下拉框，默认「自动选择」，用户可手动指定。

## 成功标准

- 自动降级成功率 > 90%
- 手动指定通道可覆盖自动选择
