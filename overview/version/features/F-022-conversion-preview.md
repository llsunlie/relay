# F-022 转换预览

**里程碑**: v0.1.1 (格式转换)

## 用户故事

作为发送方，格式转换完成后我能预览转换结果，确认无误后再发送，避免格式错乱影响接收方体验。

## 触发时机

格式转换完成后，在交付窗口的摘要栏下方显示转换预览区域。

## 预览内容

转换后的文件内容预览（纯文本渲染），文件名旁标注转换路径（如「周报.md → 周报.html」）。

## 用户操作

- 确认无误 → 点击「确认交付」发送转换后版本
- 不满意 → 点击「发送原格式」以原始格式发送
- 重新选择 → 回到交付窗口修改

## 成功标准

- 预览渲染与实际发送效果一致
- 切换格式操作 1 次点击完成
