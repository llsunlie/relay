# F-006 交付记录

**里程碑**: v0.0.1 (MVP)

## 用户故事

作为用户，我在 Agent 面板中查看所有交付历史，知道每次交付的状态，失败的可以一键重发。

## 入口

系统托盘右键菜单 →「交付记录」

## 列表展示

每条记录一行，显示交付时间、交付类型、文件名、收件人姓名、状态（已送达/发送失败）。

## 操作

- 点击某条记录展开详情：完整文件列表、留言内容、发送时间
- 失败记录行右侧有「重发」按钮，点击后重新投递
- 列表顶部有搜索框，支持按文件名和收件人姓名搜索

## 存储

本地 JSON 文件 `~/.relay/history.json`，每次交付完成后追加一条记录。

## 依赖

- F-005 邮件投递

## 成功标准

- 每次交付完成后历史列表即时更新
- 搜索和展开详情操作响应流畅
- 失败记录可重发
