# F-015 本地收件箱界面

**里程碑**: v0.1.0 (收件箱)

## 用户故事

作为用户，我从托盘打开收件箱，看到所有发给我的交付内容按处理状态分两区排列，一眼就能看出哪些还没处理。

## 入口

系统托盘右键菜单 →「收件箱」

## 布局

两区上下排列：
- 待处理：未读或尚未操作的交付
- 已归档：已读或已处理完毕的交付

## 每条收件显示

交付类型图标、文件名、发送者姓名、接收时间。

## 搜索与筛选

顶部搜索框，支持按文件名和发送者搜索。筛选按钮按交付类型和接收时间筛选。

## 依赖

- F-010 托盘状态指示

## 成功标准

- 两区布局清晰可区分
- 搜索和筛选响应流畅
