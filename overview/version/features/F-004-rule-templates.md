# F-004 规则模板

**里程碑**: v0.0.1 (MVP)

## 用户故事

作为用户，当我交付的文件名包含特定关键词时，确认弹窗中的交付类型和收件人自动预填，减少我的手动操作。

## 5 个预置规则

| 文件名关键词 | 预填交付类型 | 预填收件人角色 |
|-------------|------------|--------------|
| `周报` 开头 | 周报 | 上级 |
| `作业` 开头 | 作业提交 | 老师 |
| `请假` 开头 | 请假申请 | 上级 |
| `报销` 开头 | 报销申请 | 上级 |
| 无匹配 | 通用 | 不预填 |

## 匹配逻辑

提取文件名前缀与关键词表比对。命中后从交付对象中查找对应角色的联系人，多个同角色联系人时预填第一个。用户仍可在弹窗中手动修改预填结果。

## 依赖

- F-002 交付对象

## 成功标准

- 4 个关键词各触发正确的类型和角色预填
- 无匹配时不阻塞流程，类型默认「通用」
