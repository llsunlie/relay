# F-007 文件夹监听引擎

**里程碑**: v0.0.2 (文件夹监听)

## 用户故事

作为用户，我将文件放入 `~/.relay/outbox/` 目录后，系统自动检测到新文件并触发交付流程，我不需要手动拖拽文件。

## 监听目录

`~/.relay/outbox/`，Agent 首次启动时自动创建。

## 触发时机

文件写入完成后 2 秒内无新变更，确认文件写入完毕，系统托盘弹出通知：「检测到新文件，点击打开交付窗口」。点击通知后弹出交付窗口，文件列表自动填入，类型和收件人根据规则模板预填。

## 依赖

- F-003 交付窗口

## 成功标准

- 文件放入监听目录后 3 秒内弹出通知
- 文件系统事件不漏报
