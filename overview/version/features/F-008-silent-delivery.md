# F-008 静默交付

**里程碑**: v0.0.2 (文件夹监听)

## 用户故事

作为用户，我开启静默交付后，将文件放入 `~/.relay/outbox/` 且文件名匹配到规则模板时，系统直接发送，我只收到一条「已送达」通知。

## 触发条件

文件放入监听目录，且文件名匹配规则模板（命中交付类型和收件人角色）。

## 行为

跳过交付窗口，直接通过邮件发送。发送完成后托盘通知：「{交付类型}已自动交付给{收件人}」。

## 关闭静默

Agent 设置中可关闭静默模式，关闭后恢复默认流程（通知 → 交付窗口 → 确认发送）。

## 依赖

- F-007 文件夹监听引擎

## 成功标准

- 静默交付从文件放入到发送完成无任何窗口弹出
- 关闭静默后恢复通知→交付窗口→确认发送流程
