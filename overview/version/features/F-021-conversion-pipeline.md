# F-021 格式转换引擎

**里程碑**: v0.1.1 (格式转换)

## 用户故事

作为发送方，系统在发送前将文件转换为接收方偏好的格式，我无需手动转换。

## 触发时机

交付确认后，若接收方有格式偏好且与当前文件格式不同，自动转换后再发送。

## v0.1.1 转换路径

- Markdown → HTML、PDF、TXT
- DOCX → Markdown、PDF
- TXT → Markdown
- CSV → Markdown 表格

## 失败处理

转换失败时弹出通知「{文件名} 格式转换失败」，用户可选择「以原格式继续发送」或「取消」，取消后回到交付窗口可手动调整。

## 成功标准

- 基础转换路径可用
- 单次转换 < 3 秒
